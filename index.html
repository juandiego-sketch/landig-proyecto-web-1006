Listas ordenadas HTML

¿Que es?

Una etiqueta HTML es un fragmento de código, encerrado entre símbolos < y >, que sirve como marcador para dar instrucciones al navegador sobre cómo estructurar y mostrar el contenido de una página web.
Una lista ordenada en HTML se crea con la etiqueta contenedora
    (ordered list), utilizando
  1. (list item) para cada elemento numerado. Por defecto son numéricas, pero el atributo type permite usar letras (A, a) o números romanos (I, i). El atributo start define el número de inicio
    por ejemplo:
    1. huevos
    2. pan
    3. leche
